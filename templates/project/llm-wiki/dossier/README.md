# The dossier — structured comprehension before divergence

Understanding the problem is a phase with teeth, not a feeling. The dossier is
written ONLY through `scripts/dossier.py` and holds two kinds of entries under
a fixed ten-section pattern (object, statement, history, evidence,
neighborhood, obstructions, toolbox, cruxes, activity, formalization):

- **Questions** — the researcher's, mined from transcripts (`intake`) or added
  directly; classified as `abierta`, `pre-precisa` (not yet a mathematical
  question), `parcial`, or `respondida`. `respondida` REQUIRES >= 2 sources,
  all deep-read and current — the script refuses less.
- **Facts** — tiered `establecido` (>= 2 deep-read sources), `heuristica`
  (>= 1 read source), or `especulacion` (labeled as such). A paper nobody read
  supports nothing.

Anti-superficiality mechanisms, all enforced or measured:

1. the ten sections are mandatory — an empty section must declare its
   emptiness and why (`note`); silence is forbidden;
2. two-source rule at the top tiers, checked against the bibliography's
   reading levels (stale readings do not count);
3. adversarial audit (`audit`): an isolated worker hunts for missing
   questions, weak claims, and thin sections — findings feed the next round;
4. saturation is measured (`round`): when two consecutive rounds yield almost
   nothing new, the base is done — by data, not by feel.

`digest.md` (established facts + open cruxes + pre-precise warnings) is
injected into ideation and decomposition together with the prior-art digest,
so imagination starts FROM understanding. The digest never decides truth; the
gate remains the only judge.
