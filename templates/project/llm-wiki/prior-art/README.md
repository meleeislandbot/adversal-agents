# Prior art

Known results relevant to the target problem, each with a real citation (name,
author, year, DOI/URL). This is what stops the council from re-deriving
established theorems and calling it progress.

An entry needs a citation. "This looks familiar" is not prior art. The prior-art
auditor reads this section to ground its judgments instead of relying on a model's
(often hallucinated) memory.
