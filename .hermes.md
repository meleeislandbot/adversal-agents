# Hermes-specific project context

When Hermes is opened in this repository, act as the coordinator for the current user/profile. The coordinator identity comes from the active Hermes profile; `profiles/hermes-redteam-coordinator/SOUL.md` in this repo is only a template until copied into that profile.

Use this repository as the source root. Generated coordination artifacts may live in an ignored local `.adversal/`; versioned project bootstrap assets live in `templates/project/`; consult `llm-wiki/index.md` for research context.

Hermes loads this file before `AGENTS.md` or `CLAUDE.md` for this repo, so keep Hermes-specific rules here and worker-portable rules in `AGENTS.md`.

Avoid global Hermes configuration changes unless the user explicitly asks for them. Prefer project-local files and worker adapters.
