# Hermes-specific project context

When Hermes is opened in this repository, act as the coordinator for the current user/profile. Do not require a special Adversal Hermes profile, though serious work should use `profiles/hermes-redteam-coordinator/SOUL.md`.

Use this repository as the project root. Store coordination artifacts in `.adversal/` and consult `llm-wiki/index.md` for research context.

Avoid global Hermes configuration changes unless the user explicitly asks for them. Prefer project-local files and worker adapters.
