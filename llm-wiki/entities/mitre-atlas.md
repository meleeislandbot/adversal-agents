# MITRE ATLAS

## Overview

MITRE ATLAS is a knowledge base for adversary tactics, techniques, and case studies involving AI-enabled systems. OWASP’s agentic threat guide explicitly points readers to MITRE ATLAS among related security references. ^[raw/articles/owasp-agentic-ai-threats-and-mitigations-pdf.md]

## Relevance to Adversal Agents

ATLAS can supply standardized adversarial vocabulary for reporting red-team scenarios and mapping test cases to tactics/techniques. It should complement, not replace, agent-specific concepts like [[indirect-prompt-injection]], [[tool-use-and-privilege-risk]], and [[memory-and-rag-poisoning]].

## Caveat

ATLAS is a taxonomy/reference base; Adversal Agents still needs executable benchmarks and traces to turn taxonomy into evidence.
